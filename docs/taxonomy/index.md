# Taxonomy

The Taxonomy section defines the shared vocabulary for multi-agent system architectural
patterns. These definitions are authoritative across all project documentation.

!!! warning "Working Draft"
    The taxonomy is a working draft intended for community feedback and refinement,
    not a final specification. See [Open Challenges — Taxonomy Stabilization](../concepts/behavioral-envelope.md#open-questions).
